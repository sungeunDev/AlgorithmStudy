## ㅁ 터미널 사용법 (swift 파일 생성 / 컴파일 / 터미널로 실행)
- vi : file edit
- compile : swiftc ~.swift
- run : ./kangaroo
- run & textfile : ./kangaroo < input.txt > output.txt

- Make Linux  
: vi Makefile  
: 단축키 실행하려면 앞에 탭 꼭 넣어야 함

```
ex.

    ./kangaroo
make run (으로 실행)
```
